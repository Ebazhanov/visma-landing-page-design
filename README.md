## 🛠️ Tech Stack
- React
- TypeScript
- Vercel
- v0.app

## 📦 Getting Started

1. Clone the repo  
   `git clone https://github.com/stasandrianov9/v0-landing-page-design.git`
2. Install dependencies  
   `npm install`
3. Run locally  
   `npm start`
